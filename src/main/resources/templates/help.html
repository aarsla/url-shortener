Installation and running

Start application

java -jar shortener-0.1.0.jar

Open application in browser

http://localhost:8080

Application interaction

Resource creation

Create resources (accounts, shortcodes) using application/json requests from any of the standard REST clients.

Redirection

Visit any of created shortlinks for redirection to original registered urls.

Statistics

View documentation and generated accounts/shortcodes/stats in your favorite browser.

Recommended REST Clients

  • Rest Shell - https://github.com/spring-projects/rest-shell
  • Cocoa Rest Client - http://mmattozzi.github.io/cocoa-rest-client
  • Rest Client - http://code.google.com/p/rest-client

API

Registering new account

HTTP Method POST
URI /account
Auth Not required
Required arguments accountId
Response type application/json
Response example { success : true, description : "Your account is now active", password: "xC345Fc0" }
Response example { success : false, description : "Account already exists" }

Registering new short url

HTTP Method POST
URI /register
Auth Required, Basic
Request headers Basic authentication
Required arguments url
Optional arguments redirectType (defaults to 302, unless 301 is specifically requested)
Response type application/json
Response example {"shortcode" : "L1RHw"}
Response example { shortcode : "JwWmz", description : "ShortUrl already exists" }
Response example { description : "Malformed URL" }

Account statistics

HTTP Method GET
URI /statistic/{accountId}
Auth Required, Basic
Request headers Basic authentication
Required arguments accountId
Response type application/json
Response example { "http://www.urlone.com" : 8, "http://www.urltwo.com" : 3, "http://www.urlthree.com" : 2 }
